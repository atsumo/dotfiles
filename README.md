##[homesick](https://github.com/technicalpickles/homesick)
======


dotfilesの管理にhomesickを使用しています。

インストール方法（ruby）

	gem install homesick

設定

	homesick clone git@github.com:atsumo/dotfiles.git
	homesick list
	
シンボリックリンクをしてくれる

	homesick symlink

gitconfig.localを生成して、プライベートなものはそこに突っ込む
